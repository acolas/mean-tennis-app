Règlement :

Tie-Break = 5 points

Match non officiel à 1 set gagnant = 25 points

Match non officiel à 2 set gagnant = 50 points

Match non officiel à 3 set gagnant = 100 points

Match officiel contre un adversaire moins bien classé = 100 points

Match officiel contre un adversaire classé au même niveau = 150 points

Match officiel contre un adversaire mieux classé = 200 points
